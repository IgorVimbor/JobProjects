    Версия 1.0.1 150725

Версия исполняемого файла: main_150725.py

Изменения:

    1. В pdf_processor.py вместо бибилиотеки pdf2image использована PyMuPDF с модулем fitz.
Для работы pdf2image требуется установка Poppler на все ПК, где планируется использование приложения.
Возникли трудности с установкой Poppler на Windows 7. На Windows 10 устанавливалась poppler-24.08.0-0.
При попытке установки poppler-0.68.0 на Windows 7 возникала ошибка. Вместо использования pdf2image при
разпознавании PDF использован PyMuPDF с модулем fitz. Этот модуль не требует установку Poppler.

    2. В main.py добавлена возможность выбора файла PDF перетаскиванием (drap-and-drop).
Для этого понадобилось импортировать встроенную библиотеку tkinterdnd2.

    3. В act_processing_150725.spec явно указан импорт библиотеки tkinterdnd2, т.к. при создании файла exe
обычной командой: pyinstaller --onefile --windowed --icon=IconGreen.ico --name="act_processing_140725" main.py
библиотека tkinterdnd2 не импортировалась.
