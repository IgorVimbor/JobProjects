 1. -------- ФИНАНСОВАЯ АНАЛИТИКА ---------

1.1 Суммы претензий по потребителям:
График: столбчатая диаграмма
Оси:
  X - потребители
  Y - сумма претензий (claim_amount_all)
Фильтры: год, валюта
Дополнительно: TOP-5 потребителей

1.2 Динамика по месяцам:
График: линейный график
Оси:
  X - месяцы года
  Y - сумма претензий
Линии: выставлено vs признано
Фильтры: год, потребитель, валюта

1.3 Процент признания претензий:
График: круговая диаграмма или прогресс-бар
Показатели:
  - Выставлено: claim_amount_act
  - Признано: costs_act
  - % признания = costs_act / claim_amount_act * 100
Группировка: по потребителям

2. ----------- ВРЕМЕННОЙ АНАЛИЗ ------------

2.1 Время рассмотрения:
График: гистограмма
Расчет: response_date - claim_date (в днях)
Показатели:
  - Среднее время
  - Минимум/максимум
  - Распределение по диапазонам (0-7, 8-14, 15-30, 30+)
Группировка: по потребителям

2.2 Тренды:
График: линейный с трендом
Показывает: растет или падает количество/сумма претензий
Период: помесячно за год или по кварталам

3. ---------- ПО РЕЗУЛЬТАТАМ ПРЕТЕНЗИЙ -----------

3.1 Распределение результатов:
График: круговая диаграмма
Категории: result_claim (принято/отклонено/частично)
Дополнительно: таблица с количеством и суммами

3.2 Успешность по потребителям:
График: сгруппированная столбчатая
Оси:
  X - потребители
  Y - количество претензий
Группы: по результатам (цвет)

4. ---------- СВЯЗЬ С РЕКЛАМАЦИЯМИ ------------

4.1 Конверсия рекламация → претензия:
Показатели:
  - Всего рекламаций за период
  - Из них с претензиями
  - % конверсии
  - Средняя сумма претензии на рекламацию
Группировка: по потребителям

4.2 TOP изделий по суммам претензий:
График: горизонтальная столбчатая
Данные из связанных рекламаций (product_name)
TOP-10 изделий по суммам

5. ----------- ВАЛЮТНЫЙ АНАЛИЗ -----------

5.1 Распределение по валютам:
График: круговая диаграмма
Показывает: % претензий в каждой валюте
Таблица: количество и сумма по валютам

5.2 Конверсия в единую валюту (опционально):
Если есть курсы валют - пересчет всего в RUB
Для корректного сравнения
